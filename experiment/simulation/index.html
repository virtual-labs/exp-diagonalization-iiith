Cantor's Diagonalization
Description
  • This simulation aims to show that the set of all binary sequences is uncountable, using Cantor's Diagonalization argument
  • Since the number and length of the binary strings used is infinite, only a small part of it is visible. By choosing a value in window size, you can choose the number and length of binary strings visible.
  • In accordance with Cantor's Diagonalization argument, find a substring thats different from all the visible substrings, thus making it different compared to their parent strings.
  • Size of the substring to be entered must match the size of substrings in the string window that is visible to you. A smaller sized substring can not be different from the string in the string window with certainity. A larger sized string predicts the diagonal element of the next string which may not be correct given we can not predict the next string with certainity.
  • Check your answer using the check button. If your answer is incorrect but correct length, you will be shown the incorrect bits.
