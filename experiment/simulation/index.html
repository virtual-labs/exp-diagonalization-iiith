Cantor's Diagonalization
Description
  • This simulation shows that the set of all binary sequences is uncountable, using Cantor's Diagonalization argument
  • We assume the set of all binary sequences was countable. This means we can enumerate them.
  • Consider the set T of all infinite sequences of binary digits
  • The sequences given below in the grid in the simulation are randomly generated elements from T, then an element s of T can be constructed that doesn't correspond to any element in the enumeration.
  • A sequence s is constructed by choosing the 1st digit as complementary to the 1st digit of the first element, the 2nd digit as complementary to the 2nd digit of second element, the 3rd digit as complementary to the 3rd digit of third element, and generally for every n, the nth digit as complementary to the nth digit of nth element in the grid
  • By construction, s is a member of T that differs from each element in the grid, since their nth digits differ. Hence, s cannot occur in the enumeration.
  • With s being different from all other enumerations by definition,T does not contain all possible enumeration of binary sequences. Note that this argument works for every enumeration. Hence, it is not possible to enumerate the set of all binary sequences which implies set of all binary strings is uncountable.
